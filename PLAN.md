# JR Fitness Website — Project Plan

## Overview
A client-side Angular SPA for a personal training business. Goal is to learn Claude Code while maintaining hands-on input over UI/design decisions.

## Stack
- **Framework:** Angular (client-side only, no SSR)
- **Design source:** Stitch (MCP-connected for reference)

---

## Page Structure

Single-page application with scroll-based sections. Two hero CTA buttons pan the user to specific sections on the page.

### Sections (top to bottom)

| # | Section | Notes |
|---|---------|-------|
| 1 | **Hero** | Header + two buttons: "Apply for Coaching" (scrolls to offerings) and "View Results" (scrolls to transformations) |
| 2 | **Methodology** | Text/content block explaining the training philosophy |
| 3 | **Offerings** | Card-based layout of coaching packages/services |
| 4 | **Client Transformations** | Before/after or progress photos |
| 5 | **Footer** | Google Maps embed for training area + social media icon links |

---

## Design
- Designs created in Stitch — MCP connected for direct reference during implementation
- UI/design decisions are owner-driven; implementation assistance via Claude Code

---

## Development Approach
- Owner retains final say on all UI/design choices
- Claude Code assists with implementation, structure, and code quality
- Iterative: design → build section by section → review → refine
